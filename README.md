# Financial Analyser

A full-stack financial tracking application that allows users to categorise transactions and analyse spending patterns through interactive dashboards.

## Tech Stack
- Python
- Streamlit
- SQL
- FastAPI

## Features
- Track and categorise financial transactions
- Interactive dashboard for spending analysis
- Backend API for data processing
- Secure data storage using a relational database

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## Use Case
This application helps users gain insights into their spending behaviour by visualising financial data in real time.
